	GIT : Git BASH

> open Git Bash as admin
> navigate to folder with cd e:/folder

> create a new branch
> git branch MyBranch
> git checkout MyBranch

> git checkout master
> now we are back to the main branch

> push MyBranch as the main branch
> git merge MyBranch
